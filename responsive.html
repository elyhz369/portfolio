REDESIGN: Toronto Cupcake


Redesigning the webpage for a bakery store.


Have you ever visited a website and find it really hard to navigate? A good website should be user-centric with the following properties: First, new users should easily understand what to do (learnability). In addition, it should be intuitive to recall how to use and what to do (memorability). Futhermore, typical tasks should be done quickly (efficiency). Keeping these qualities in mind, this project will focus on redesigning a website and making it more user friendly.



PART I - Identifying Problems


The site I picked is Toronto Cupcake. I chose this site because I used it to order cupcakes for my friend, and the website design was notoriously known in my friend group. The problems are as follows:


  1. LOW USABILITY
    • Navigation bar located at the bottom with a very small font size. It’s hard for users to even notice it.
    • Inconsistent hierarchy. Delivery time, cupcake ingredients, and branding slogan should not be placed in the same box.
